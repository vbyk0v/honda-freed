# Памятка по ТО Honda Freed

## Модели

- **GB5** — бензин + вариатор, 2WD  
- **GB6** — бензин + вариатор, 4WD  
- **GB7** — гибрид + робот, 2WD  
- **GB8** — гибрид + робот, 4WD  

## Полезные материалы

- Альбом / памятка:  
  [ПАМЯТКА ПО ТО HONDA FREED (Drive2)](https://www.drive2.ru/l/665720756587477593/)

## Моторное масло

### Требования
- API **SN / ILSAC GF-5**  
- или **API SP / ILSAC GF-6**

### Объём
- **3,3 литра** (при замене)

### Рекомендуемые масла
- **Honda 08227-99974** — Ultra LEO-SP 0W-20  
- **Moly Green 470101** — Hybrid SN/GF-5 0W-20
